Overview

Why Thermal Stress Detection Matters

Thermal comfort directly impacts pig health, growth rate, and welfare. Existing monitoring systems rely on manual inspection or single-modality sensing, which limits scalability and consistency under varying lighting and environmental conditions. This work investigates whether combining visual (RGB) and thermal (IR) information improves classification of pig thermal stress states.

🌡️

Welfare Monitoring

Detects thermal distress early, enabling timely environmental adjustments to ventilation and heating systems before health deteriorates.

📷

Dual Modality

Fuses standard RGB video with infrared thermal imagery; RGB captures posture and appearance while IR reveals thermal signatures invisible to the eye.

⚙️

Cross-Validation Strategy

Performance is evaluated using 5-fold cross-validation, where each fold contains data from multiple recording weeks to assess generalisation.

🔬

Training Protocol

The training setup includes balanced class sampling, spatial augmentation, gradient clipping, and learning rate scheduling to improve stability and reproducibility.

Training

Training Pipeline

The model is trained using 5-fold cross-validation. Each fold is trained independently with early stopping, learning rate scheduling, and gradient clipping based on validation performance.

01
Split by Fold

Train on 4 folds (~3,994 samples), validate on the held-out fold (~999 samples). No data leakage between splits.

02
Augmented DataLoader

Training set applies spatial + colour augmentation. Validation uses clean resize-only transforms.

03
Train + Validate Loop

Each epoch: forward pass → loss → backprop → grad clip → optimizer step. Validation loss drives scheduler and early stopping.

04
Save Best Checkpoint

Best model per fold saved to pig_stress_model_fold{N}.pth based on lowest validation loss.

05
Early Stopping

Training halts if val loss doesn't improve for 5 consecutive epochs, preventing overfitting on a plateau.
